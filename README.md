### Installation
   
1. Install Composer, PHP and NPM

2. Install Composer Packages
   ```sh
   composer install 
   ```
3. Install NPM Packages
   ```sh
   npm i 
   ```
4. Set DB_* env variables to your database

5. Run Migrations
   ```js
   php artisan migrate
   ```
6. Run Server
   ```js
   composer run dev
   ```
